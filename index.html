color-robin

A JavaScript library for extracting the colors from an image.

Showcase

Code example

HTML 

    <div>
        <img src="image.jpg" id="image" />
    </div>

JavaScript 

    import { analyzeImage } from 'color-robin';

    const image = document.getElementById('image');
    
    analyzeImage(image).then(histogram => {
        console.log(histogram.getColors(5));
    });
